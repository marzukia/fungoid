# Fungoid Theme for VSCode

A dark theme with orange and green dominant colors, matching the Fungoid TUI aesthetic.

## Colors

- **Background**: Dark grey (#1e1e1e)
- **Foreground**: Warm white (#e8e6e3)
- **Green**: #3cdc50 (primary accent)
- **Orange**: #ff7800 (primary accent)
- **Yellow**: #ffbe00 (secondary)
- **Red**: #ff5a5a (errors)
- **Muted**: #695541 (secondary)

## Installation

1. Copy the `fungoid-vscode-theme` folder to your VSCode extensions directory
2. Reload VSCode
3. Select "Fungoid" from the color theme picker (Ctrl+K Ctrl+T)

## Usage

```json
{
  "workbench.colorTheme": "Fungoid"
}
```
